# Expense Tracker - Трекер расходов

**Автор:** Абдрашитов Тимур

## Описание программы

Приложение для отслеживания личных расходов с графическим интерфейсом на Python (Tkinter). Позволяет:

- Добавлять расходы с указанием суммы, категории и даты
- Просматривать список всех расходов в таблице
- Фильтровать расходы по категории и дате (периоду)
- Подсчитывать общую сумму расходов за выбранный период
- Получать статистику по категориям
- Удалять расходы
- Сохранять и загружать данные в формате JSON

## Как использовать

### Установка и запуск

1. Установите Python 3.6 или выше
2. Скачайте файл `expense_tracker.py`
3. Запустите командой:
   ```bash
   python expense_tracker.py
